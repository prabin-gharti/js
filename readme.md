## ðŸ’» What is JavaScript (JS)?

JavaScript (JS) is one of the three core technologies of the World Wide Web, alongside **HTML** (HyperText Markup Language) and **CSS** (Cascading Style Sheets).

Think of a webpage like a **human body**:

* **HTML** is the **skeleton** (the structure and content).
* **CSS** is the **skin and clothing** (the styling and appearance).
* **JavaScript** is the **muscles and nervous system** (the *actions* and *interactivity*).

---

## âš¡ What Does JS Do?

Essentially, JavaScript makes websites **dynamic** and **interactive**. It allows you to add *behavior* to static HTML content.

### Key Capabilities of JS:

* **Interactivity:** It handles what happens when a user clicks a button, scrolls, or types something.
    * *Example:* Making a **"dark mode"** switch work.
* **Dynamic Content:** It can change, add, or remove HTML content and CSS styles *after* the page has loaded.
    * *Example:* Updating the time on a digital clock or showing a new image in a **photo carousel**.
* **Form Validation:** It checks if a user has filled out a form correctly *before* sending the data to a server.
* **Backend Development (with Node.js):** While it started in the browser, JS can also be used on the server side to build the backend logic of an application.

---

## ðŸ§­ Where Does JS Run?

1.  **In the Browser (Client-Side):** This is its original and primary role. Every major web browser (Chrome, Firefox, Safari, Edge) has a **JavaScript Engine** (like Google's V8 engine) that executes the code.
2.  **On the Server (Server-Side):** Using an environment like **Node.js**, JavaScript can run outside the browser to handle databases, user authentication, and application logic.
